
## 常用元字符

```
1. .匹配出换行符以外的任意字符
2. \w 匹配字母或数字或下划线
3. \s 匹配任意的空白字符
4. \d 匹配任意数字
5. \n 匹配一个换行符
6. \t 匹配一个制表符
7. 
8. ^ 匹配字符串的开始
9. $ 匹配字符串的结尾
10. \W 匹配非字母或数字或下划线
11. \D 匹配非数字
12. \S 匹配非空白字符
13. a|b 匹配字符a或者b
14. （）匹配括号内的表达式，也表示一个组
15. [...] 匹配字符组中的字符
16. [^...] 匹配除了字符组中的字符的所有字符
```

## 量词

```
1. * 重复0次或多次
2. + 重复一次或多次
3. ？重复0次或一次
4. {n} 重复n次
5. {n,} 重复n次或更多次
6. {n,m} 重复n次到m次
```

## 贪婪 非贪婪
```
1.  .* 贪婪匹配
2. .*? 非贪婪匹配
```
